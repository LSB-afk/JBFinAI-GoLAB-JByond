# 04_증빙

대회 요구사항, 공식 자료, 기사 근거, 정책 근거를 추적하는 공간입니다.

## 증빙 원칙

- 공식 자료를 우선 연결합니다.
- 기사 근거는 pain point를 설명하는 보조 근거로 사용합니다.
- Agent 판단과 연결되는 출처는 Evidence Feed에서 확인 가능해야 합니다.
