# 발표 5분 시나리오 전략

> **⚠️ 제약(공식 [[본선_안내문]] 정정)**: **예선(1·2차 ROUNDING) = 5분 발표 + 2분 Q&A**(전 팀) / **최종 발표평가(상위 5팀) = 15분 발표 + 5분 Q&A**. 제출 마감 **7/5 10:30**(현장 9:00 실작업 마감). 발표 시 반드시 시연 가능.
> 여러 접근을 비교해 고른 뒤 리허설. 🎨 Figma: 시나리오 비교(node 4801:2)·5분 런시트(node 4802:2).
> 무기: 전주 카페 히어로 · 키보드 UX("일은 에이전트가, 사람은 눈만") · 확장성(계열사) · 로컬모델·PII 비반출 · 김민주 지역 공감 · MVP 실동작.
> **평가 5축×20 + 시연영상 가점5 = 105점**: ①주제적합·문제정의 ②금융연계·고객가치 ③AI Agent 설계·구현(판단·행동·검증) ④MVP 완성도·검증 ⑤혁신성·확장성·리스크. (기존 "4축" 이해는 오류 — [[본선_안내문]] §5)

## 시나리오 (여러 개)
- **S1 데모-퍼스트** ⭐예선 — 훅 30s→라이브 데모 3분(전주 카페 골든패스)→차별화 1분→클로징 30s. 강점: MVP·완결성 최강. 리스크: 데모 실패 → 폴백 녹화 필수.
- **S2 스토리-퍼스트** ⭐결선 — 페인 스토리 1분(김민주 지역공감·부모 보이스피싱)→데모 2.5분→신뢰·보안 1분→비전 30s. 강점: 혁신성·공감. 리스크: 데모 시간 압박.
- **S3 SCQA 논리** — 문제·복잡성→질문→답(데모)→증명(확장성·보안·리서치). 강점: 사업화·논리. 리스크: 설명적.
- **S4 평가기준 정조준** — **5축×20**(주제적합·문제정의 / 금융연계·고객가치 / AI Agent 설계·구현 / MVP 완성도 / 혁신성·확장성·리스크) 정조준. 강점: 채점 직결. 리스크: 감동 약함.
- **S5 1-메시지 집중** — 슬로건 "눈으로 일한다"로 관통. 강점: 각인·차별화. 리스크: 깊이 부족.

## ⭐ 추천 (2라운드 전략)
- **예선(Round 1/2, 생존)**: S1 데모-퍼스트 + 강한 훅(S5 슬로건) → 완결성·MVP로 확실히 상위5 진입.
- **결선(상위 5, 우승)**: S2 공감 아크 얹기 → 김민주 지역 스토리로 차별화.

| 문제 | 0:30–1:00 | 자료 흩어짐·업무량·신뢰/보안 장벽 | 김민주 |

## 🎙️ Q&A 대비 (예선 2분 / 결선 5분)
1. 실제 은행 데이터 없이? → 공공API + 합성 페르소나, 실연동 설계 제시.
2. AI 오판단 책임? → 사람 최종승인·감사체인, 에이전트는 보조.
3. 계열사 확장 실현성? → 역할별 대시보드 = 계열사 공통 구조(조직도로 증명).
4. 로컬 컴퓨팅 비용? → 온프렘 로컬모델·PII 비반출(보안+TCO 근거 D19/D23).
