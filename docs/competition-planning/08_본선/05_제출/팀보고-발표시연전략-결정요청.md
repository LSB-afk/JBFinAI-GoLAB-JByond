# 📣 팀 보고 — 발표·시연영상 스토리 전략 (결정 요청)

> **읽는 분께**: 7/2·7/3 **회의 녹취 원본을 처음부터 끝까지 다시 읽고** 발표와 시연영상 스토리를 다각도로 짰습니다. 아래 판단이 **논리·데이터로 납득되면 택1**해 주세요. 그대로 진행하겠습니다. 상세 근거·후보 전체는 → [[발표-시연영상-스토리전략-심화]].
> **한 줄**: *발표는 "설득", 영상은 "완성도 증명" — 따로 짠다. 우리는 은행 전문가 코스프레 대신 '기회를 주면 이 속도로 AX를 만드는 젊은 실행자'로 간다.*

---

## 1. 회의에서 우리가 이미 합의한 것 (재확인)

- 계열사 **전북은행 + JB우리캐피탈 2개**, 구동 **하이브리드(로컬 실동작·이승보 PC)**, **키보드 퍼스트 UX**, **사람 최종승인·PII 비반출**. ✅
- 실동작 도메인 **전세사기 vs 보이스피싱 택1**은 **7/4 오전 확정 예정**(나머지 더미). → **이 전략은 둘 중 무엇이 와도 그대로 성립**하게 짰습니다.

## 2. 원본에서 다시 건진 핵심 판단 4가지

**① 발표는 "잘함"으로 점수를 안 준다. 산출물이 본체다.**
심사 5축(각 20점)에 "발표력" 항목이 없습니다(7/3 38:23). 김민주 말대로 *"잘 만든 놈이 이기는 게임."* → **발표자료 PDF·기능명세서·GitHub·데모의 완성도**에 힘을 싣고, 발표는 그 채점 축을 **효율적으로 전달**하는 도구로 씁니다.

**② MVP가 진짜 돌아가면 상위권 바닥이 깔린다.**
7/2에서 나온 판단: *"모델이 실제로 돌아가고 산출물이 신뢰할 만하면 3등 이상 가능."* 배점이 전부 20점 균등이라 **한 축도 못 버립니다.** → 발표·영상 모두 **로컬모델 실동작을 눈으로** 보여줍니다(말로 하지 않음, 7/3 15:45).

**③ "우리는 은행을 모른다"가 약점이 아니라 포지셔닝이다.**
데이터도 인터뷰도 없이 은행 내부를 다 알 순 없습니다. 그런데 이 대회는 **채용 퍼널**이고(7/3 50:12 *"인재를 찾기 위해… 거르고 거르는 지능형 서비스실"*), *"기회를 주면 어떻게 할 수 있냐"*를 봅니다(7/2 01:34). → **"빠른 실행자"로 당당하게** 갑니다. 전문가 흉내 금지.

**④ 공감은 김민주가 캐리한다 — 진짜 경험으로.**
7/2 01:02에서 나온 얘기 그대로입니다: 전북/전라도 정서에 **먹힐 사람은 김민주**. 게다가 김민주는 **본인이 5,000만원 청년 전세대출 당사자**(7/2 01:16) — 연기가 아니라 **1인칭 실화**로 공감을 만들 수 있습니다. 논리는 김주용, 기술 증거는 이승보가 맡습니다.

## 3. 추천 (택1 해주세요)

| 결정 | 추천안 | 왜 |
|---|---|---|
| **예선 발표(5분+2분)** | **데모-퍼스트** + 슬로건 훅 + "우리는 빠른 실행자" 1줄 | 완성도·주제적합 직결로 상위 5팀 확실히 진입 |
| **결선 발표(15분+5분)** | **공감 아크**(김민주 실화) + 5축 빠짐없이 커버 | 차별화·잠재력으로 우승 다툼 |
| **시연영상(가산 5점·5분)** | **"제품 설명서"** 백본 + 30초 인간 훅 + **인터넷 끊고 로컬모델 실동작 컷** | 필수요소 빠짐없이 = 완성도 점수 + 기억점 |
| **발표자** | 김민주=스토리·공감·클로징 / 김주용=데모 해설·논리·확장성 / 이승보=라이브 조작·로컬 증명 | 정서·논리·증거 분리, 3회 발표 체력·리스크 분산 |
| **슬로건** | **"손은 놓고, 눈만"** (제품명은 추후) | 회의에서 나온 표현 직계·각인 |

> 후보는 서로 배타적이지 않습니다. 위는 **가장 안전하면서 각인되는 조합**입니다. 다른 조합을 원하시면 심화문서 §3·§4의 P1~P5 / V1~V4에서 골라 주세요.
