## 2. PDF 생성 명령 (복붙 가능)

```bash
cd "08_본선/03_제품/07_발표-제출"
# 개별 변환: node _패키징/md-to-pdf.mjs <입력.md> <출력.pdf>
node _패키징/md-to-pdf.mjs 기능명세서.md   _패키징/기능명세서.pdf
node _패키징/md-to-pdf.mjs MVP제안서.md    _패키징/MVP제안서.pdf

# 검증: 비어있지 않은지 + 페이지 수 (pdfinfo 없으면 파일크기 >50KB로 대체 판단)
pdfinfo _패키징/기능명세서.pdf | grep -E "Pages|Page size"   # A4 · 페이지수 확인
ls -la _패키징/*.pdf                                          # 50KB 이상이면 정상

# 한글 두부(tofu) 의심 시: --html 로 중간 HTML 저장 후 브라우저로 육안 확인
node _패키징/md-to-pdf.mjs 기능명세서.md _패키징/_check.html --html && open _패키징/_check.html
```

- 스크립트는 playwright chromium `page.pdf()` 사용(루트 `node_modules`에 설치됨, 새 의존성 無).
- 폰트: Pretendard → Apple SD Gothic Neo → Noto Sans KR 폴백. 표 테두리·배경 인쇄됨(`printBackground`).
- `_패키징/*-draft.pdf`는 **드래프트**(문서 변경 가능성). 문서 확정 후 위 명령으로 재생성.
