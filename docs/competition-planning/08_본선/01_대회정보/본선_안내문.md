## 1. 대회 개요

### 배경

JB금융그룹 Fin:AI Challenge는 금융과 인공지능의 융합을 통해 미래 금융의 새로운 해답을 함께 만들어갈 인재를 찾기 위해 마련되었습니다.

금융 현장에서는 단순한 업무 자동화를 넘어 고객의 상황을 이해하고 필요한 판단을 내리며 그에 맞는 행동을 수행하는 지능형 서비스에 대한 수요가 점차 커지고 있습니다.

JB금융그룹은 실제 금융 비즈니스와 연결될 수 있는 AI Agent 서비스의 가능성에 주목하며, 금융의 현재를 개선하고 미래의 고객 경험을 새롭게 설계할 창의적이고 실행력 있는 아이디어를 발굴하고자 합니다.

### 대회 방식

오프라인 본선 — 예선에서 제출한 MVP를 기반으로 서비스를 한층 더 발전시키고 고도화하는 단계입니다. 본선 진출팀은 **1박 2일** 동안 서비스 완성도를 높이고, 최종적으로 심사위원 앞에서 발표 및 최종 시연을 진행합니다.

**장소**: JB금융그룹 아우름캠퍼스 (전북 정읍)  
**일정**: 7월 4일(토) ~ 5일(일) 1박 2일

### 주최 / 운영

| 구분 | 기관 |
|------|------|
| 주최 | JB금융그룹 |
| 운영 | 데이콘 |

## 6. AI Agent 공통 요건

본선 제출 서비스는 AI Agent 서비스로서 아래 3요소를 반드시 포함해야 합니다.

| 구분 | 내용 |
|------|------|
| **판단** | 입력, 상황, 데이터 등을 해석하여 상태, 위험, 우선순위 등을 도출 |
| **행동** | 판단 결과에 따라 다음 행동을 선택 (예: 추가 질문, 근거 조회, 수정안 생성, 알림, 요약, 예약 등) |

## 10. 대회 규칙 및 유의사항

### 10-1. 본선 운영 관련

- 본선 제출물: 발표자료·기능명세서·GitHub 링크 (시연영상은 선택)
- 본선에서는 예선 제출물을 기반으로 고도화한 서비스를 발표해야 하며, 심사위원 앞에서 구현한 서비스의 주요 기능과 동작 흐름을 **직접 시연**해야 함

### 10-2. 개발 환경 관련

- 개발에 필요한 컴퓨팅 리소스는 **별도로 제공되지 않음**
- 오프라인 본선에서도 **참가자 본인의 장비**로 개발 및 시연이 가능해야 함

### 10-3. 기능명세서 관련 유의사항

- 기능명세서는 참가팀이 직접 기획·구현한 서비스의 기능을 기준으로 작성
- 타 팀 또는 타인의 결과물·기능 구성·문서 내용·기능명세서를 무단 복제·도용·사용 시 실격
- 본선 진출팀은 **예선에 제출한 기능명세서를 기준 문서로 관리**해야 함
- 개발 과정에서 기능의 추가·수정·삭제 등 변경 발생 시 기능 변경이력 항목에 반드시 기재
- 기능 변경이력은 심사 과정에서 기획·개발 연속성, 구현 경과, 산출물 간 정합성 확인에 활용됨

### 10-4. 경진대회 일반 유의사항

- 제출 기한과 제출 방법을 반드시 숙지, 기한 내 모든 산출물 제출
- 제출물 중 일부가 누락되거나 제출 형식이 맞지 않을 경우 평가 대상에서 제외 가능
- **'팀장'으로 등록된 사용자에게만 '제출 완료' 버튼이 활성화**되어 최종 제출 가능
- 각 단계별 최종 제출 후 **수정 불가** — 반드시 확인 후 최종 제출
- 부정 행위 금지; 확인 시 평가 제외·본선 진출 취소·수상 취소 등 조치

### 10-5. 제출 유의사항

- 제출 자료는 아이디어 유출 방지와 내부 평가를 위해 **비공개**로 게시
- 발표자료와 기능명세서는 반드시 **PDF 형식**으로 제출
- 제출 자료에는 안내된 **필수 작성 항목을 반드시 반영**
- 운영진은 필요 시 GitHub 링크·코드·데모 링크·기타 보완자료 추가 제출을 요청할 수 있으며, 참가팀은 안내된 기한과 방식에 따라 제출해야 함

## 11. 실격 요건 (8개)

다음 각 호에 해당하는 경우 본선 참가 또는 심사 대상에서 제외될 수 있습니다.

1. 사전에 안내한 **필수 제출물을 기한 내 제출하지 않은** 경우
2. 본선 현장에서 심사위원 대상 **직접 시연이 불가능한** 경우
3. 참가 신청 내용 또는 본선 진출 자격과 관련하여 **허위 사실이 확인**된 경우
4. 대회 운영 규정, 참가 규정, 저작권 또는 부정행위 관련 **규정을 위반**한 경우
5. 타 팀 또는 타인의 결과물·코드·문서·기능명세서 등을 **무단으로 복제·도용·사용**한 경우
6. 기능 변경이 발생하였음에도 **기능 변경이력에 기재하지 않거나 허위로 작성**한 경우
7. 본선 발표 또는 심사 진행이 불가능할 정도로 **무단 불참하거나 운영 요청에 불응**한 경우
8. 오프라인 본선 운영 장소의 **필수 준수사항을 중대하게 위반**한 경우 (예: 음주, 무단 이탈, 대회 운영 방해 등)
