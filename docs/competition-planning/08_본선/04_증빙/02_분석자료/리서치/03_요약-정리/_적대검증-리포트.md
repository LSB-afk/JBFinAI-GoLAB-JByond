# 적대검증 리포트

## 검증 범위

- 검증 방식: 오프라인 회수본 교차검증. 웹 검색, 외부 조회, 원문 재확인은 수행하지 않음.
- 기준선: `_canon.md`
- 입력 회수본: 지정 결과 디렉토리의 대상 파일 전체. 지정 세트에서 파일 없음 항목은 발견되지 않음.
- 판정 원칙: 회수본 사이 충돌은 해소하지 않고, 일치·발산·단일소스·`_canon` 충돌 후보로 분리함.
